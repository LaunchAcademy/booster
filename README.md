# Booster (Launch Academy's Rails Template)

* clone this repo

    ```
      git clone git://github.com/LaunchAcademy/booster.git
    ```

* create a rails app from this template

    ```
      rails new <app_name> -m <path_to_repo>/template.rb
    ```
